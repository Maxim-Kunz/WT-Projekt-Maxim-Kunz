# WT-Projekt-Maxim-Kunz

## Projektbeschreibung
Dieses Projekt ist eine Web-Anwendung zur Protokollierung und Auswertung von Krafttrainingseinheiten. Der Fokus liegt auf einer simplen, robusten Erfassung von Übungen (z.B. Dips, Liegestütze, Klimmzüge), Sätzen und Wiederholungen. Das Projekt entsteht im Rahmen des Moduls "Web-Technologien" an der HTW Berlin.
